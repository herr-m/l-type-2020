# L-type (2020)
Vertical scrolling shoot'em up / space shooter game written in c++17.

## Features
- 2 players coop
- Scoreboard
- Level editor
- Level browser (sharing)
- Social (friendslist, messaging, authentication)
- Ternial HUD (Ncurses) and graphical HUD (SFML)

## Dependencies
- ncurses
- SFML
- Boost serialization
- Sqlite3
